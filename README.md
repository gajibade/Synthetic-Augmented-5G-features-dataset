Initial public release of the Synthetic Augmented 5G Features Dataset. The synthetic 5G features were obtained from the Wednesday data of the CICIDS2017 dataset.
After preprocessing by removing entire rows with infinity or not a number (NaN), we added 5G-specific features: network latency, signal strength, and user equipment
(UE) throughput to each record. Following the prepressing approach highlighted above, the dataset is free of overfitting, data leakage, or strong feature-target
correlations and hence ready for immediate use. We provide both the CSV file and the RAR file for convenient download and usage. Please cite the dataset and the 
supporting paper whenever you make use of it.

The dataset can also be found in https://doi.org/10.5281/zenodo.20363569
